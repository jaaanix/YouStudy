PHP-Zugriff auf die MySQL-Datenbank
 
Fr einen Connect per PHP bentigen Sie die Angabe des MySQL-Servers. Dieser lautet bei STRATO rdbms.strato.de (anstelle von "localhost").

Als Passwort tragen Sie Ihr Passwort ein, welches Sie nach dem Anlegen der Datenbank im Kundenservicebereich selbst vergeben haben.

Ein Beispielskript, wie ein Verbindungsaufbau in PHP aussehen knnte, finden Sie hier. Tragen Sie dabei bitte jeweils hinter db_benutzer, db_passwort sowie db_name die entsprechenden Angaben Ihrer Datenbank zwischen den Anfhrungszeichen ein.
 
# Zugangsdaten
$db_server = 'rdbms.strato.de';
$db_benutzer = 'U2152898';
$db_passwort = '';
$db_name = 'DB2152898';
# Verbindungsaufbau
if(mysql_connect($db_server, $db_benutzer, $db_passwort)) {
echo 'Server-Verbindung erfolgreich, whle Datenbank aus...
';
if(mysql_select_db($db_name)) {
echo 'Datenbank erfolgreich ausgewlt, alle Tests abgeschlossen.';
}
else {
echo 'Die angegebene Datenbank konnte nicht ausgewhlt werden, bitte die Eingabe prfen!';
}
}
else {
echo 'Verbindung nicht mglich, bitte Daten prfen!
';
echo 'MYSQL-Fehler: '.mysql_error();
}
?>
